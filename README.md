# EcoBot

O EcoBot é um aplicativo/site desenvolvido com o objetivo de auxiliar os usuários no descarte correto de resíduos, contribuindo para a conscientização ambiental e para a promoção de práticas sustentáveis.

## Problema

O descarte incorreto de resíduos é um problema recorrente na sociedade atual. Muitas pessoas não possuem conhecimento adequado sobre a separação correta do lixo, o que resulta em impactos ambientais negativos, como poluição do solo, da água e do ar.

## Proposta de solução

O EcoBot propõe uma solução tecnológica que utiliza recursos de identificação e classificação de resíduos para orientar o usuário sobre a forma correta de descarte. O sistema permite a entrada de informações por texto ou voz, fornecendo respostas de forma simples e objetiva.

## Funcionalidades

- Identificação de resíduos por texto ou voz  
- Classificação em categorias: reciclável, orgânico, rejeito e perigoso  
- Orientações sobre descarte adequado  
- Indicação de pontos de coleta (em desenvolvimento)  
- Interface simples e intuitiva  

## Tecnologias utilizadas

- Flutter (aplicativo mobile) e/ou HTML, CSS e JavaScript (versão web)  
- Lógica de classificação de resíduos  
- Interface focada em usabilidade
